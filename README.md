# gain-exposure

A skill for [Claude Code](https://claude.com/claude-code) (and compatible agent harnesses) that turns *"promote my project"* into a real multi-platform launch instead of one Reddit post and a shrug.

## What it does

Most AI agents, when told to promote a project, do this:

1. Pick one subreddit.
2. Post.
3. Stop.

That is not a launch. This skill makes the agent do what an actual founder does on launch day: post across **many** subreddits, run a Twitter thread, submit Show HN, write an IndieHackers launch, a Dev.to deep-dive, a LinkedIn story post, short-form video on TikTok / YouTube Shorts / Instagram Reels, and hit newsletters / Discord / Slack communities — all from **your own logged-in browser**, without leaking keys or tripping bot detection.

## How it works

- Launches [Chrome DevTools MCP](https://github.com/ChromeDevTools/chrome-devtools-mcp) with `--autoConnect` so the agent reuses your existing Chrome profile. Your Reddit / Twitter / LinkedIn / TikTok logins come along. No scraping credentials, no headless-browser fingerprints.
- **Reasons visually from screenshots** instead of `document.querySelector`. Screenshot → locate form controls → click coordinates → type with human cadence (50–150ms jitter). This defeats the bot detection that catches DOM-level form fills.
- **Autonomous with escalation**: runs the full target list end-to-end, skips on failure, loops back once at the end to retry, escalates only what fails twice.
- **Preflight gates** before any posting: landing-page OG card check, UTM link registry, per-subreddit rule scan (flair / day-of-week / karma gate / link rules).
- **Image assets** generated by driving ChatGPT or Gemini image mode via the same Chrome session — iterate in chat until the image is right, save to a dedicated launch-assets folder. Uses your existing Plus / Advanced subscription instead of paid APIs.
- **Video lane** (optional) with Remotion for the hero sizzle, ElevenLabs for voiceover, and any clipping tool (Opus Clip / Submagic / Klap / Vizard) for auto-cut shorts.

## Install

Clone into your Claude Code skills directory:

```bash
git clone https://github.com/Co-Messi/claude-skill-gain-exposure.git \
  ~/.claude/skills/gain-exposure
```

Or symlink if you want to track updates:

```bash
git clone https://github.com/Co-Messi/claude-skill-gain-exposure.git \
  ~/code/claude-skill-gain-exposure
ln -s ~/code/claude-skill-gain-exposure ~/.claude/skills/gain-exposure
```

### Prerequisites

- Claude Code (or any agent harness that loads skills from `~/.claude/skills/`)
- Node.js (for `npx chrome-devtools-mcp@latest`)
- Chrome 144+ (required for `--autoConnect`)
- You logged into the platforms you want to post on, in your normal Chrome profile

### Optional (for video lane)

- ElevenLabs API key (voiceover)
- A Remotion skill / installation (hero sizzle render)
- A clipping-tool subscription (Opus Clip / Submagic / Klap / Vizard / Descript)
- ChatGPT Plus or Gemini Advanced (for image gen via web UI — bypasses paid image APIs)

## Usage

In Claude Code, after you ship a project:

```
/gain-exposure
```

or just say:

> Promote OpenHedge across every platform where quant traders live.

The skill will:

1. Run an **intake** (asks you the project, angle, audience, target URL, tone, credentials, social-proof prime list).
2. **Launch Chrome DevTools MCP** and verify you're logged in on every target platform. Halts on any platform that isn't logged in — you log in manually in the attached Chrome.
3. **Preflight**: OG card check on your landing page, UTM link registry, Reddit per-sub rule scan (marks subs GO / DEFER / SKIP).
4. **Produce assets** (images via Gemini/ChatGPT web chat; video via Remotion + TTS if requested).
5. **Post everywhere** autonomously — skip on failure, retry at the end, escalate only what fails twice.
6. **Report back** with every post URL in a clean launch report.

## Philosophy

- Cast a wide net. Don't stop at one subreddit. Don't skip video if the project has a visual surface.
- Behave like a human using their eyes. No DOM manipulation, no instant form fills, no millisecond-perfect clicks.
- Ask before you hardcode. API keys, tool choices, social-proof lists — always confirm with the user rather than assuming.
- Post-and-forget. The agent's job ends when posting is done. Monitoring and replying to comments is the user's job.
